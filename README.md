## How to dev

First you need to install NodeJS and NPM.

1. Install electronjs:

   ```console
   $ npm install -g electron
   ```

2. Install all dependencies, in project root directory: 
	
	```console
   $ npm install
   ```

3. Set dev environment variable: NODE_ENV=dev

4. Start app:

   ```console
   $ npm start
   ```
